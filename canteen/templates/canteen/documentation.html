How to add a dish?

So, we're cookin'! And you want to display the dishes. Well, here's how to do that.

  1. Go to the admin site.

  2. Locate the "Dishes" link, and at its right, click the "Add" link that has the plus icon.

  3. Fill in the name, price, description, and select the photo of the dish from your device (make sure that you ready that photo). The sold out check box in not mandatory, you'll only check it if the canteen runs out of that dish.

  4. Finally, save the dish in the database. You have three options of saving it.

    • Save - Simply save it, and the site will redirect you to the list of all dishes.

    • Save and add another - Just like the simple saving, but it will redirect you to the same page with the form being cleaned.

    • Save and continue editing - It's simply saving the dish as well, but it will redirect you to the same page containing the form that has the values you've entered. Prefered if you want to further edit the dish.
