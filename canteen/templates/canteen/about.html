About the website

ESTI Canteen Online Ordering System is an art of the ICT-12 students below. This application makes the ordering process much faster. Students no longer have to wait in line. Also, students do not need to go down stairs just to know the foods being sold at the canteen, thus, saving them time and energy. This website also features a user-friendly design. The developers made the interface as simple as possible to give the users a fine experience.

The purpose of this web application is to provide a new way of how to order dishes at the canteen. Its role is to serve as the e-commerce website of the ESTI canteen. The programmers were motivated to create this web application when they found it very tiring to still go downstairs just to know what dishes are being sold at the canteen. They are the ICT-12 students whose room is located on the top of the building. This web application is a practical system to come up with.

Deployment process

Create a Heroku account. An email is needed to create a Heroku account.

Install Python 3.7.1 and Git. Python is the programming language that is mostly used by the website in the server. While Git is a control version system that records the code changes made by the programmers. Git is also used for pushing the web app's code to the Heroku server.

Install the Heroku Command Line Interface and login using the terminal command $ heroku login.

Prepare the app using the terminal command $ pip freeze > requirements.txt, and a text file will be created at the root directory of the app.

Create the app on Heroku, which prepares Heroku to receive your source code. Do this using the terminal command $ heroku create esticanteen.

Do the final commit in git. This would now include the requirement.txt file that was generated by Python for you. Do the terminal command $ git commit -am "Final".

Finally, push the web app's source code on Heroku using the terminal command $ git push heroku master. Wait for it to finish, and pray that the web app will build successfully.

The web application is now accessible around the world.
